# Data Files

This directory contains the evaluation datasets for the IApred antigenicity predictor.

## External Evaluation Data

- `External_evaluation_antigens.csv`: 218 antigen sequences for external validation
- `External_evaluation_non_antigens.csv`: 218 non-antigen sequences for external validation

## Training Data

The training data consists of 918 antigen and 918 non-antigen sequences from various pathogens (bacteria, viruses, fungi, protozoa, helminths).
